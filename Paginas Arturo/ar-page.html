Aprende de páginas web

En esta lección verás como la función en HTML de como hacer que se muestre y se oculte la contraseña para un formulario.


Paso 1: Crear el Campo de Contraseña

Comenzaremos creando un campo de entrada con tipo password, que oculta el texto por defecto. Añadimos también un botón para alternar la visibilidad.

<form>
  <label for="contra">Contraseña:</label>
  <input type="password" id="contra" placeholder="Escribe tu contraseña">
  <button type="button" id="MostrarContraseña">Mostrar</button>
</form>

Paso 2: Escribir el Código JavaScript

A continuación, añadimos un script para alternar entre mostrar y ocultar la contraseña. Aquí está el código:

// Seleccionamos los elementos del DOM
const passwordField = document.getElementById('contra');
const toggleButton = document.getElementById('MostrarContraseña');

// Agregamos un evento 'click' al botón
toggleButton.addEventListener('click', () => {
  // Cambiamos el tipo del campo de 'password' a 'text' y viceversa
  const type = passwordField.getAttribute('type') === 'password' ? 'text' : 'password';
  passwordField.setAttribute('type', type);

  // Cambiamos el texto del botón
  toggleButton.textContent = type === 'password' ? 'Mostrar' : 'Ocultar';
});
